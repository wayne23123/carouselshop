# 您好，歡迎來到 Wayne Lu 的商品列表練習

這個商品列表使用 HTML + CSS + VUE 製作而成

概念是在 CSS 用 V-bind 綁定一個 ref 值 並宣告一個 moveCarouselRight 函示
